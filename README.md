markdown# QualyTools

Plataforma de Ferramentas da Qualidade com IA real (Claude Sonnet).

## Stack
- **Frontend:** HTML/CSS/JS puro → Vercel
- **Backend:** Node.js + Express → Railway
- **Pagamentos:** Mercado Pago Subscriptions
- **IA:** Anthropic Claude Sonnet

## Setup Local

### Frontend
Abra `frontend/index.html` direto no navegador.

### Backend
cd backend
cp .env.example .env
# edite o .env com suas credenciais
npm install
npm run dev

## Deploy
- Frontend → Vercel (pasta `frontend`)
- Backend → Railway (pasta `backend`)
